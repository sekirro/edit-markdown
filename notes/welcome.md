# 📓 欢迎使用 Markdown Notebook

这是一个简洁优雅的 Markdown 笔记本，支持：

## ✨ 功能特性

- **Markdown 渲染** — 实时预览，支持 GFM（表格、任务列表等）
- **代码高亮** — 代码块自动语法高亮
- **分屏编辑** — 编辑/预览/分屏三种模式切换
- **自动保存** — 修改后 3 秒自动保存，也可手动 Ctrl+S
- **全文搜索** — Ctrl+F 快速搜索所有笔记内容
- **文件夹管理** — 新建文件夹、重命名、删除

## 🚀 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存当前笔记 |
| `Ctrl+F` | 打开搜索 |
| `Tab` | 缩进 |
| `Shift+Tab` | 取消缩进 |

## 📝 Markdown 示例

### 代码块

```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

print(list(fibonacci(10)))
```
